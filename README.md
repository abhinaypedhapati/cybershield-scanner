# 🛡️ CyberShield - Vulnerability Scanner

A professional vulnerability scanner that detects security weaknesses in networks and web applications.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Port Scanning** | Detects open ports (21,22,23,25,53,80,443,445,3306,3389) |
| ⚠️ **Weak Configurations** | Finds FTP, Telnet, SMB, HTTP vulnerabilities |
| 🌐 **Web Scanning** | Detects SQL Injection & XSS attacks |
| 📊 **Report Generation** | Exports JSON reports with severity levels |
| 🎨 **Modern UI** | Clean dark theme with smooth animations |
